컴퓨터는 고차미분 방정식을 풀수 없으므로 일차미분방정식을 푸는형태로 나타낸다

n차 미분 방정식 -> n개의 state 를 만들어 낸다 ex) 2차 -> x1, x2

그리고 각 상태를 벡터 형태로 나타낸다

상태 방정식 -> x 미분 = Ax + Bu
